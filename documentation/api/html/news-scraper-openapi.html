Create a new profile

Request Body schema: application/json
required
name
required
string
description
required
string
useCustomSources
required
boolean
Default: false

When true, user-provided URL and RSS sources are required. When false, API stores empty source arrays and AI-recommended sources are used.

tags
required
Array of strings
roles
required
Array of strings

Optional role labels for profile metadata and background scraping context. Duplicate names are rejected case-insensitively.

required
Array of objects (ProfileUrl)

Required with at least 1 entry only when useCustomSources=true.

required
Array of objects (RssSettings)

Required with at least 1 entry only when useCustomSources=true.

notificationProfileId
integer or null

Backward-compatible single notification profile id. When notificationChannelIds is provided, this value is normalized to the first selected channel id.

notificationChannelIds
Array of integers[ items >= 1 ]

Optional multi-select notification channel ids associated with the profile. Every id must exist in notification profiles; unknown ids return 400.
